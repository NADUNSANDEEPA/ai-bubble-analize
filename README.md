# ai-bubble-analize
Dot-com vs AI Company Financial Comparison

The workbook contains four sheets:

Summary: period definitions, top-10 company lists, aggregate tables and comparison charts.
DotCom_Data: 1995–2002 company-by-company observations.
AI_Data: 2022–2025 company-by-company observations.
Methodology: definitions, limitations and source references.

Period used
Dot-com cycle: 1995–2002.
AI market boom: 30 November 2022 to the present. ChatGPT’s public release on 30 November 2022 is used as the starting point because it accelerated mass-market generative-AI adoption. The cycle has not ended, so no final burst date can yet be identified.
The spreadsheet uses 2022–2025 completed annual observations. It excludes 2026 because the current year is incomplete.
Measurements included

Each company table provides:

year;
company and ticker;
listing status;
year-end market value or private valuation;
year-over-year market-value growth;
revenue;
revenue growth;
profit or earnings proxy;
profit growth;
profit-measure basis;
data-completeness status;
source URLs.
